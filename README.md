# Симулятор муравейника на Lua &amp; LÖVE ( Love2D )

Есть два муравейника: один желтый, другой синий. В каждом сидит муравьиная королева, которая пытается выжить врага со своей территории

Мастерская №07, Летняя школа юных программистов, 2016 г.

## Функциональность

* 2 муравейника
* Работает под любой ОС, где есть рантайм LÖVE
* Масштабирование с помощью колеса мыши.

## Требования

512Mb RAM, OpenGL 2.0+, OS Windows / Linux / Mac OS / Android

## Запуск

Установите LOVE ( Скачать можно с http://love2d.org ).
Запускайте с помощью drag-and-drop папки проекта в LOVE.EXE.
Либо c:\ваш путь до LOVE\love.exe c:\путь папки до проекта >
